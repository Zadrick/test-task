# test-task

## Project TT
```
ТЗ верстка
Необходимо подготовить шаблон дашборда,
из шапки, левого меню и рабочей области.
В шапке слева должен быть логотип (любой),
в центре строка поиска с выпадающем списком при клике, в правой части
кнопка выхода.
состоящего из левого меню с пунктами
1. Главная

2. Сайт

3. Заявки

4. Настройки

В рабочей области должны быть хлебные крошки и под ними сам блок в
котором будет название текущего раздела и счетчик с количеством
переходов в текущий раздел, так же нужно подключить любой плагин с
всплывающем notify справа сверху, которое будет выводить текст текущего
раздела каждый раз при переходе в него.
При переходе по разделам Главная, Сайт, Заявки в рабочей области должны
меняться хлебные крошки.
При переходе в раздел Настройки, должно меняться Layout, в нем должно
поменяться левое меню на пункты Группы пользователей, Пользователи.
В шапке вместо строки поиска кнопка вернуться к публичной части.
Для счетчика использовать Vuex.
Для html использовать pug.
Для стилей использовать scss.
Для роутинга и хлебных крошек vue-router
```
